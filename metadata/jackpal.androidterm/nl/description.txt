Open de ingebouwde Linux command shell in Android en laat uw innerlijke geek los!

Dit is een nieuwe versie van de populaire "Android Terminal Emulator" applicatie. Hetzelfde goede programma, alleen met een nieuwe naam.

Top features

+ Volledige Linux Terminal emulatie. 
+ Meerdere vensters.
+ Launcher snelkoppelingen.
+ UTF-8 tekst. (Arabisch, Chinees, Grieks, Hebreeuws, Japans, Koreaans, Russisch, Thais, etc.)
+ volledig gratis, geen advertenties, Geen in-app-aankopen, Geen opdringerige vensters, niets.

Snelle VGV:

+ Als je moeite hebt met tekst invoeren, probeer dan de gratis app "Hacker's Keyboard IME" te installeren. (Specifiek als je een Samsung of HTC apparaat hebt.)
+ Deze app is geen spel-emulator.
+ Deze app helpt je niet om je telefoon te 'rooten' of om de IMEI van je telefoon te veranderen.
+ Je moet weten (of willen leren) hoe je de linux command-line werkt om deze app te kunnen gebruiken.
+ Je kunt bijvoorbeeld command-line functies installeren zoals "Busy Box" of "Debian chroot" als je meer commands wil gebruiken dan de commands die standaard ge√Øncludeerd zijn in je android-apparaat.
+ Als je een -505 error krijgt als je deze app op Android 5.0 probeerd te installeren, betekend dat dat een andere app dezelfde toestemmingen gebruikt. Als je terminal-emulator voor android will installeren, moet je dan eerst de andere app de-installeren. (Het schijnt dat "jrummy's Toolbox Pro" dit probleem veroorzaakt.)
