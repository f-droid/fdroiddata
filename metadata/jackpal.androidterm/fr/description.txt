Accédez à l’interpréteur de commandes Linux intégré d’Android.

Nouvelle version de la populaire application « Android Terminal Emulator ». Le même programme sous un nouveau nom.

Fonctions principales

+ Émulation complète du terminal Linux.
 + Multiples fenêtres.
 + Raccourcis du launcher.
 + Texte UTF-8 (arabe, chinois, grec, hébreu, japonais, coréen, russe, thaï, etc.)
 + Totalement gratuit. Sans pubs, achats, fenêtres pop-up, rien.

FAQ rapide :

+ Si vous éprouvez des difficultés avec la saisie de texte, essayez d’installer le logiciel gratuit « Hacker's Keyboard IME ». (Particulièrement si vous avez un appareil Samsung ou HTC.)
+ Cette appli n’est pas un émulateur de jeux. 
+ Cette appli ne vous aidera pas à débrider votre téléphone ni à changer son IIEM. 
+ Pour cette appli, vous devez savoir (ou vouloir apprendre) comment utiliser la ligne de commande de Linux.
+ Vous pourriez installer un ensemble d’utilitaires en ligne de commande tels que « Busy Box » ou « Debian chroot » afin d’ajouter des commandes qui ne sont pas les commandes intégrées à votre appareil Android. 
+ Si vous obtenez une erreur -505 en essayant d’installer cette appli sous Android 5.0, cela signifie qu’une autre appli utilise les mêmes droits. Vous devrez désinstaller l’autre appli afin d’installer « Terminal Emulator » pour Android. (Certaines personnes me disent que « jrummy's Toolbox Pro » cause ce problème.)
