Pencarian buku, produk dan web dikirim ke peladen Google (tergantung dari pilihan negara asal), meskipun tombol pencarian khusus bisa dibuat di pengaturan seperti duckduckgo.com/?q=%s atau wikibooks.org/w/index.php?search=%s.

Mendukung tipe barcode berikut:

* UPC-A dan UPC-E
* EAN-8 dan EAN-13
* Kode 39; 93; 128
* ITF
* Codabar
* RSS-14 (semua variasi)
* Kode QR
* Data Matriks
* Aztec (kualitas 'beta')
* PDF 417 (kualitas 'alpha')

Kunjungi situs web untuk pemecahan masalah dan kejelasan mengenai perizinan. Aplikasi ini mengijinkan Anda untuk membagi kontak, aplikasi, dan bookmark dalam Kode QR. Inilah kenapa perizinan Kontak diperlukan. Lihat "Kunjungi Situs Web Pengembang" di bawah ini ( https://github.com/zxing/zxing/wiki/Frequently-Asked-Questions ).

Jika perangkat Anda tidak memindai, coba dulu pemecahan masalah untuk bug perangkat di Pengaturan. Aktifkan semuanya, dan kemudian coba nonaktifkan satu per satu untuk menentukan yang diperlukan. Jika ini tidak membantu, coba hapus cache dan pengaturan perangkat dari pengaturan Android.

Jika ini tetap tidak bekerja dan anda percaya ada masalah dengan aplikasinya, kirimkan pesan ke pengembang aplikasi dengan informasi berikut. Pesan tanpa informasi ini tidak akan ditindak lanjuti, akan diabaikan.

- Tipe perangkat
- Deskripsi dasar tentang apa yang tidak bekerja: crashes? tidak bisa memindai? terlihat aneh?
- Pengakuan bahwa anda telah mencoba langkah di atas terlebih dahulu
