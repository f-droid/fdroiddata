Scannez des codes barres sur les produits, ou des Data Matrix, ou des Qr codes, contenant des URL (site web), des contacts, etc.

Les recherches sur le web, des produits, des livres passent par un serveur Google (selon le choix du pays d'origine), malgré la création d'un bouton de recherche dédié dans les paramètres, i.e. duckduckgo.com/?q=%s ou wikibooks.org/w/index.php?search=%s.

Supporte les types de code-barres suivants :

* UPC-A et UPC-E
* EAN-8 et EAN-13
* Code 39 ; 93 ; 128
* ITF
* Codabar
* RSS-14 (toutes les variantes)
 * QR Code
 * Data Matrix
* Aztec (qualité 'beta')
* PDF 417 (qualité 'alpha')

Consultez le site Web pour le dépannage et des explications sur les autorisations. L’appli vous permet de partager des contacts, des applis et des marque-pages dans un code QR. C’est pourquoi l’autorisation Contact est nécessaire. Voir le site « Visit Developer Website » ci-dessous (https://github.com/zxing/zxing/wiki/Frequently-Asked-Questions).

Si votre appareil ne scanne pas, essayez d'abord des méthodes alternatives. Dans paramètres, activez toutes les options possibles, puis désactivez les l'une après l'autre pour trouver celles qui sont nécessaires. Si cela ne fonctionne toujours pas, videz le cache et les paramètres de l'appareil dans les paramètres Android.

Si celles-ci ne fonctionnent pas et que vous pensez qu'il s'agit d'un problème de l'application, envoyez un message à l'auteur avec les informations suivantes. Les messages sans ces informations ne pourront pas être traités, et seront donc ignorés.

- Type d’appareil
 –Une description sommaire de ce qui ne fonctionne pas : plantages ? Ne numérise pas ? Drôles d’apparence ?
 –Confirmation que vous avez d’abord essayé les étapes ci-dessus
– Une capture des journaux de l’appareil quand il démarre
