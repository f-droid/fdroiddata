Summary:Plugin for Transdroid
Description:
Transdroid Torrent Search is a reusable Android application that merely
provides torrent search results (links to .torrent files) from various web
sources. These results are exposed via a ContentProvider that multiple Android
apps can use to search for torrents while maintaining control over the search
interface.

See also [[org.transdroid.full]].
