Summary:Nautical charts plugin
Description:
Plan navigation in the open sea and inland waterways. The
charts can contain various information like sailing routes,
navigation lights, dangerous areas, areas where it's
permitted to sail or dock, etc. Uses the
[http://www.openseamap.org/ OpenSeaMap] project which is
based on OpenStreetMap.

This plugin requires [[net.osmand.plus]] version 2.0.0+.
